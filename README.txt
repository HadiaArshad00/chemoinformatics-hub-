DEPLOYMENT ON NETLIFY
--------------------------------------------------------------------------------
Method 1: Drag & Drop (Easiest)
  1. Extract the ZIP file to a folder
  2. Go to https://app.netlify.com/drop
  3. Drag and drop the folder onto the Netlify page
  4. Your site will be live instantly

Method 2: Git Repository
  1. Create a new GitHub/GitLab repository
  2. Upload the 4 files to the repository root
  3. Go to https://app.netlify.com/ > Add new site > Import existing project
  4. Connect your Git provider and select the repository
  5. Click "Deploy site"

RUNNING LOCALLY
--------------------------------------------------------------------------------
  python -m http.server 8000
  # Then open http://localhost:8000 in your browser

KEYBOARD SHORTCUTS
--------------------------------------------------------------------------------

  Escape     Close navigation menu / glossary tooltip
